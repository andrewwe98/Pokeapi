

A React + Vite application for exploring Pokémon data.

## Features

- Browse Pokémon 
- Search and filter functionality
## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Technologies

- React
- Vite
- [PokéAPI](https://pokeapi.co/)
