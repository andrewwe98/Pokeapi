

A React + Vite application for exploring Pokémon data.

## Features

- Browse Pokémon by generation
- View detailed stats and abilities
- Search and filter functionality
- Fast, responsive UI with Vite

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Technologies

- React
- Vite
- [PokéAPI](https://pokeapi.co/)
